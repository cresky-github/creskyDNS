## 🚀 立即开始

### 第一步：阅读快速开始指南
👉 [QUICK_START.md](QUICK_START.md)

### 第二步：查看快速参考
👉 [DOMAIN_LIST_QUICK_REF.md](DOMAIN_LIST_QUICK_REF.md)

### 第三步：创建你的域名列表
创建一个 `my_domains.txt` 文件：
```text
google.com
example.com
```

### 第四步：验证文件格式
```bash
python validate_domain_lists.py my_domains.txt
```

### 第五步：配置应用
编辑 `config.yaml`：
```yaml
lists:
  custom:
    path: "my_domains.txt"
```

### 第六步：启动应用
```bash
cargo run
```
